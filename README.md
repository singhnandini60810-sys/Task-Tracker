# Task Tracker
A productivity tracker with task scheduling, streaks, reminders and heatmap analytics.

## Features
• Add multiple tasks  
• Edit and delete tasks  
• Monthly tracker for each task  
• Reminder system  
• Task repetition scheduling  
• Streak counter  
• Level progression system  
• Weekly analytics dashboard  
• Yearly productivity heatmap  
• 12-color activity intensity system  

## Technologies Used
HTML  
CSS  
JavaScript  
LocalStorage
